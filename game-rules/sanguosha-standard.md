# 《三国杀》标准版完整规则与全卡牌/武将图鉴

## 一、 游戏基础规则

### 1. 身份与胜利条件
标准局通常为 8 人游戏，身份配置为：1主公，2忠臣，4反贼，1内奸。
* **主公 (Lord)**：消灭所有的反贼和内奸，平定天下。
* **忠臣 (Loyalist)**：保护主公存活，不惜一切代价协助主公消灭反贼和内奸（胜利条件与主公相同）。
* **反贼 (Rebel)**：杀死主公。
* **内奸 (Renegade)**：在除主公外其他所有人（包括忠臣、反贼和其他内奸）死后，最后单挑杀死主公，成为最后的生还者。

### 2. 回合流程
每个玩家的回合严格按照以下六个阶段进行：
1.  **准备阶段**：处理部分武将技能（如甄姬的“洛神”、诸葛亮的“观星”）。
2.  **判定阶段**：处理延时类锦囊（如“乐不思蜀”、“闪电”）。若判定区有多张延时锦囊，按“后放置的先判定”原则结算。
3.  **摸牌阶段**：从牌堆顶摸两张牌（除非有武将技能干预，如周瑜“英姿”或许褚“裸衣”）。
4.  **出牌阶段**：可以使用任意数量的牌，但需遵守以下两个基本限制：
    * 每个出牌阶段只能使用一张【杀】（除非装备【诸葛连弩】或有张飞“咆哮”等技能）。
    * 任何角色的判定区里不能同时存在两张同名的延时类锦囊。
5.  **弃牌阶段**：检查手牌数。如果手牌数超出你的手牌上限（通常等于你当前的体力值），则需要弃置超出部分的手牌。
6.  **结束阶段**：处理部分武将技能（如貂蝉的“闭月”）。

## 二、 全武将及技能说明（标准版 25 名）

### 🛡️ 魏国 (7名)
* **曹操 (4血 / 主公)**
    * 【奸雄】：你可以立即获得对你造成伤害的牌。
    * 【护驾】(主公技)：当你需要使用或打出【闪】时，你可以请求魏势力角色打出一张【闪】。
* **司马懿 (3血)**
    * 【反馈】：每当你受到一次伤害后，你可以获得伤害来源的一张牌。
    * 【鬼才】：在任意角色的判定牌生效前，你可以打出一张手牌代替之。
* **夏侯惇 (4血)**
    * 【刚烈】：每当你受到一次伤害后，你可以进行一次判定，若结果不为红桃，则伤害来源须弃置两张手牌或受到你造成的1点伤害。
* **张辽 (4血)**
    * 【突袭】：摸牌阶段，你可以放弃摸牌，改为获得最多两名其他角色的各一张手牌。
* **许褚 (4血)**
    * 【裸衣】：摸牌阶段，你可以少摸一张牌；若如此做，你本回合内使用【杀】或【决斗】造成伤害时，此伤害+1。
* **郭嘉 (3血)**
    * 【天妒】：在你的判定牌生效后，你可以获得此牌。
    * 【遗计】：每当你受到1点伤害后，你可以观看牌堆顶的两张牌，将其中一张交给任意一名角色，然后将另一张交给任意一名角色（也可交给自己）。
* **甄姬 (3血)**
    * 【倾国】：你可以将一张黑色手牌当【闪】使用或打出。
    * 【洛神】：准备阶段，你可以进行判定，若结果为黑色，你获得此牌，然后你可以重复此流程，直到判定结果为红色为止。

### ⚔️ 蜀国 (7名)
* **刘备 (4血 / 主公)**
    * 【仁德】：出牌阶段，你可以将任意数量的手牌交给其他角色，若你于此阶段内给出的牌张数达到两张或更多，你回复1点体力。
    * 【激将】(主公技)：当你需要使用或打出【杀】时，你可以请求蜀势力角色打出一张【杀】。
* **关羽 (4血)**
    * 【武圣】：你可以将一张红色牌当【杀】使用或打出。
* **张飞 (4血)**
    * 【咆哮】：锁定技，你于出牌阶段内使用【杀】无数量限制。
* **诸葛亮 (3血)**
    * 【观星】：准备阶段，你可以观看牌堆顶的 X 张牌（X为全场存活人数，最多为5），然后以任意顺序放回牌堆顶或牌堆底。
    * 【空城】：锁定技，若你没有手牌，你不能成为【杀】或【决斗】的目标。
* **赵云 (4血)**
    * 【龙胆】：你可以将【杀】当【闪】、【闪】当【杀】使用或打出。
* **马超 (4血)**
    * 【马术】：锁定技，你计算与其他角色的距离-1。
    * 【铁骑】：当你使用【杀】指定目标后，你可以进行判定，若结果为红色，该角色不能使用【闪】。
* **黄月英 (3血)**
    * 【集智】：每当你使用一张非延时类锦囊牌时，你可以摸一张牌。
    * 【奇才】：锁定技，你使用锦囊牌无距离限制。

### 🏹 吴国 (8名)
* **孙权 (4血 / 主公)**
    * 【制衡】：出牌阶段限一次，你可以弃置任意张牌，然后摸等量的牌。
    * 【救援】(主公技)：锁定技，吴势力角色对处于濒死状态的你使用【桃】时，你额外回复1点体力。
* **甘宁 (4血)**
    * 【奇袭】：你可以将一张黑色牌当【过河拆桥】使用。
* **吕蒙 (4血)**
    * 【克己】：若你于出牌阶段内未使用或打出过【杀】，你可以跳过你的弃牌阶段。
* **黄盖 (4血)**
    * 【苦肉】：出牌阶段，你可以失去1点体力，然后摸两张牌。
* **周瑜 (3血)**
    * 【英姿】：摸牌阶段，你可以多摸一张牌。
    * 【反间】：出牌阶段限一次，你可以令一名其他角色选择一种花色，然后该角色获得你的一张手牌并展示之，若此牌花色与其所选不同，你对该角色造成1点伤害。
* **大乔 (3血)**
    * 【国色】：你可以将一张方块牌当【乐不思蜀】使用。
    * 【流离】：当你成为【杀】的目标时，你可以弃置一张牌，将此【杀】转移给你攻击范围内的一名其他角色（该角色不能是此【杀】的使用者）。
* **陆逊 (3血)**
    * 【谦逊】：锁定技，你不能成为【顺手牵羊】和【乐不思蜀】的目标。
    * 【连营】：每当你失去最后的手牌时，你可以摸一张牌。
* **孙尚香 (3血)**
    * 【结姻】：出牌阶段限一次，你可以弃置两张手牌，令你和一名已受伤的男性角色各回复1点体力。
    * 【枭姬】：每当你失去装备区里的一张牌后，你可以摸两张牌。

### 🐺 群雄 (3名)
* **华佗 (3血)**
    * 【急救】：你的回合外，你可以将一张红色牌当【桃】使用。
    * 【青囊】：出牌阶段限一次，你可以弃置一张手牌，令一名角色回复1点体力。
* **吕布 (4血)**
    * 【无双】：锁定技，你使用的【杀】需两张【闪】才能抵消；你使用的【决斗】需两张【杀】才能抵消。
* **貂蝉 (3血)**
    * 【闭月】：结束阶段，你可以摸一张牌。
    * 【离间】：出牌阶段限一次，你可以弃置一张牌，令两名男性角色进行【决斗】（由你决定哪名角色先出【杀】）。

## 三、 全卡牌图鉴与说明

### 1. 基本牌 (Basic Cards)
* **【杀】**：出牌阶段限使用一次。指定攻击范围内的一名其他角色，若其无法打出【闪】，则受到1点伤害。
* **【闪】**：在你的回合外使用（通常是对【杀】的响应）。抵消一张【杀】的效果。
* **【桃】**：
    * 出牌阶段对自己使用，回复1点体力（体力已满时不能使用）。
    * 当有角色（包括自己）处于濒死状态（体力值为0或以下）时打出，使其回复1点体力。

### 2. 锦囊牌 (Stratagem Cards)
**非延时类锦囊（打出即结算）：**
* **【无中生有】**：出牌阶段对自己使用。从牌堆顶摸两张牌。
* **【过河拆桥】**：出牌阶段对除你以外的任意一名角色使用。弃置其装备区、判定区或手牌中的一张牌（若选弃手牌，需随机抽取）。
* **【顺手牵羊】**：出牌阶段对距离为1的一名其他角色使用。获得其装备区、判定区或手牌中的一张牌（若选拿手牌，需随机抽取）。
* **【决斗】**：出牌阶段对除你以外的任意一名角色使用。由目标角色先开始，你与其轮流打出一张【杀】。未打出【杀】的一方受到另一方造成的1点伤害。
* **【借刀杀人】**：出牌阶段，对除你以外的、装备区里有武器牌的一名角色使用。令其对你指定的一名其他角色使用一张【杀】。若该角色不出【杀】，则你获得其装备区里的武器牌。
* **【南蛮入侵】**：出牌阶段对所有其他角色使用。按行动顺序结算，目标角色需打出一张【杀】，否则受到你造成的1点伤害。
* **【万箭齐发】**：出牌阶段对所有其他角色使用。按行动顺序结算，目标角色需打出一张【闪】，否则受到你造成的1点伤害。
* **【五谷丰登】**：出牌阶段对所有角色使用。从牌堆顶亮出等同于现存角色数量的牌，然后按行动顺序，每名角色从中挑选并获得一张。
* **【桃园结义】**：出牌阶段对所有角色使用。所有已受伤的角色回复1点体力。
* **【无懈可击】**：在目标锦囊牌对目标角色生效前使用。抵消该锦囊牌对该角色产生的效果。

**延时类锦囊（放置于判定区，回合判定阶段结算）：**
* **【乐不思蜀】**：出牌阶段对除你以外的任意一名角色使用（放置于其判定区）。在其判定阶段，进行一次判定：若判定结果不为红桃，则跳过其该回合的出牌阶段；若为红桃，则无事发生。结算后该牌弃置入弃牌堆。
* **【闪电】**：出牌阶段对自己使用（放置于自己判定区）。在判定阶段，进行判定：若判定结果为黑桃 2~9 之间，则受到3点雷电伤害，并将此牌弃置；否则，将此牌移动到下一名角色的判定区。

### 3. 装备牌 (Equipment Cards)
装备牌放在角色的装备区。每个角色最多只能同时装备一把武器、一件防具、一匹+1马和一匹-1马。装备同类牌时，旧装备需直接弃置。

**武器 (Weapons) - 增加攻击距离并附带技能：**
* **【诸葛连弩】 (攻击距离 1)**：出牌阶段，你可以使用任意数量的【杀】。
* **【雌雄双股剑】 (攻击距离 2)**：当你使用【杀】指定一名异性角色为目标后，你可以令其选择：弃置一张手牌，或让你摸一张牌。
* **【青釭剑】 (攻击距离 2)**：锁定技，每当你使用【杀】时，无视目标角色的防具。
* **【青龙偃月刀】 (攻击距离 3)**：当你使用的【杀】被【闪】抵消时，你可以立即对相同的目标再使用一张【杀】。
* **【丈八蛇矛】 (攻击距离 3)**：当你需要使用或打出【杀】时，你可以将两张手牌当一张【杀】使用或打出（若两张牌花色一红一黑，则此【杀】视为无色）。
* **【贯石斧】 (攻击距离 3)**：当你使用的【杀】被【闪】抵消时，你可以弃置两张牌（手牌或装备区的牌均可），强制令此【杀】依然造成伤害。
* **【方天画戟】 (攻击距离 4)**：若你使用的【杀】是你出牌阶段打出的最后一张手牌，你可以为这把【杀】指定最多三名目标。
* **【麒麟弓】 (攻击距离 5)**：当你使用【杀】对目标角色造成伤害时，你可以弃置其装备区里的一匹坐骑（+1马或-1马皆可）。

**防具 (Armor)：**
* **【八卦阵】**：每当你需要使用或打出一张【闪】时，你可以进行一次判定：若判定结果为红色，则视为你使用或打出了一张【闪】；若为黑色，你仍可正常打出【闪】。

**坐骑 (Mounts)：**
* **【防御马 / +1马】**（绝影、的卢、爪黄飞电）：锁定技，其他角色计算与你的距离时，始终 +1。（使你更难被攻击到）。
* **【进攻马 / -1马】**（赤兔、大宛、紫骍）：锁定技，你计算与其他角色的距离时，始终 -1。（使你更容易攻击到别人）。
